Introduction - Setup - Using NetManager - Command Reference - Change Log


Command Referance


additem
Adds an item to character
Usage: additem CharacterName Warehouse ItemID Amount EnchantLevel Eroded Blessed Ident Wished
addskill
Adds a skill to target character
Usage: addskill CharacterName SkillID SkillLevel
banchar
Ban character
Usage: banchar CharacterName BanHour
renamechar
Renames a character
Usage: renamechar CharacterName NewName
changeclanowner
Changes clan owner
Usage: changeclanowner ClanName NewOwnerCharacterName
transchar
Transfers character to another account
Usage: transchar CharacterName AccountID
transchar2
Transfers character to another account and renames it
Usage: transchar2 CharacterName NewCharacterName AccountID
createpet
Creates a pet in characters inventory...? Untested
Usage: createpet CharacterName PetType
delbookmark
Deleted a bookmark from character
Usage: delbookmark CharID Bookmarkname
delchar
Instantly deletes a character (non-GM chars deleted in this manner are recoverable)
Usage: delchar CharacterName
delcomment
Deletes a comment, second variable is unknown and untested
Usage: delcomment CommentID option
delclan
Deletes a clan
Usage: delclan ClanName ---
kick
Disconnects a character from the server
Usage: kick CharacterName
sendhome
Sends a character to the nearest town
Usage: sendhome CharacterName
setbdracc
Creates or changes the level of a builder account
Usage: setbuilderaccount AccountName BuilderLevel
setbldrchar
Sets the builder level of a character
Usage: setbuildercharacter CharacterName BuilderLevel
setcharloc
Changes the location of a character
Usage: setcharlocation CharacterName WorldID xLoc yLoc zLoc
loginannounce
Adds a message to the login announcements.
Usage - loginannounce action(Default 1 or higher) announceid announcemessage
timedannounce
Dont use
Usage: action Interval AnnounceID AnnounceMessage
setskillall
Give all skills to target character for current class
Usage: setskillall CharacterName
