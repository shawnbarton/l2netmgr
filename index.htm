Introduction

Lineage 2 netmanager is an application which directly interfaces with the "official" Lineage 2 server software allowing more administration options than what would normally be possible. netmanager also allows administrators to perform nearly any required task without the need to load up a game client and also eliminates the risk of others infiltrating your service by only internally connecting to the designated ports.
